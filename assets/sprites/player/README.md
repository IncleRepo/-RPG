# Player Sprite Set

기본 주인공 프리셋 `미라진 여명 인양사`의 모션 프레임 세트입니다.

- 출력 크기: `48x48`
- 저장 위치: `assets/sprites/player/`
- 생성 기준: `src/scripts/player-sprite-data.mjs`
- 렌더러/PNG 생성: `src/scripts/player-sprite-renderer.mjs`, `tools/generate-player-sprites.mjs`

포함 모션:

- `idle`
- `walk`
- `run`
- `jump`
- `fall`
- `attack`
- `hurt`
- `death`

이 PNG 세트는 웹 프레임 애니메이터에서 수정 가능한 리그 데이터를 기준으로 생성됩니다.
정식 픽셀아트 리워크 전의 프로토타입 산출물로 사용합니다.
