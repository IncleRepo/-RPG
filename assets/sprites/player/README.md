# 주인공 도트 모션 세트

`반향해 연대기`의 플레이어 캐릭터 `여명 인양사`를 기준으로 제작한 기본 모션 세트입니다.

## 콘셉트 요약

- 영원한 황혼에 젖은 `등외항 미라진` 출신 인양사
- 짙은 남청 방수 외투, 바랜 청록 스카프, 소금빛 회색 장비 벨트
- 공명 능력이 드러날 때 손등과 목선에 새벽빛 금선이 번지는 실루엣
- 이동과 전투에 모두 연결하기 쉬운 측면 액션형 비율

## 에셋 규격

- 프레임 크기: `48x64`
- 배경: 투명 PNG
- 기준 방향: 오른쪽을 바라보는 측면 포즈
- 저장 위치: `assets/sprites/player/`

## 모션 구성

- `idle`: 4프레임
- `walk`: 6프레임
- `run`: 6프레임
- `jump`: 4프레임
- `fall`: 4프레임
- `attack`: 6프레임
- `hurt`: 3프레임
- `death`: 6프레임

미리보기는 `player-motion-preview.png`에서 한 번에 확인할 수 있습니다.

## 파일 규칙

모든 파일은 `motion_00.png` 형식으로 정리했습니다.

- 예시: `idle_00.png`, `walk_03.png`, `attack_05.png`

## 재생성

에셋은 아래 명령으로 다시 생성할 수 있습니다.

```bash
node tools/generate-player-sprites.mjs
```
