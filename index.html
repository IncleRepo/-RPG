2D Browser RPG

기본 이동 프로토타입

간단한 스테이지 위에서 키보드로 좌우 이동과 점프를 테스트할 수 있는 초기 플레이 화면입니다. 브라우저에서 바로 물리 감각과 조작 반응을 확인할 수 있도록 구성했습니다.

플레이 영역

입력 대기 중
상태 대기
좌표 x: 96 / y: 0

조작법

  • A / D 또는 ← / → 로 이동
  • W / ↑ / Space 로 점프
  • 플랫폼 위에 착지할 수 있으며, 낙하 중에는 중력이 적용됩니다

이번 범위

스테이지
하늘, 지면, 발판으로 구성된 단일 테스트 맵
플레이어
좌우 이동, 점프, 착지 판정을 가진 간단한 캐릭터
물리
중력, 최대 낙하 속도, 경계 제한을 포함한 기본 루프
