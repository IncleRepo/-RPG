기본 이동 스테이지

간단한 스테이지와 캐릭터를 HTML, CSS로 구성하고 JavaScript로 좌우 이동, 점프, 중력을 적용한 초기 프로토타입입니다.

플레이 테스트

A / D 또는 방향키로 이동하고 W / Space / ↑로 점프합니다.

초기화 중

조작 방법

  • A / D 또는 ← / →: 좌우 이동
  • W / Space / ↑: 점프
  • 플랫폼 위로 착지하면 다시 점프 가능
  • 화면 밖으로는 나가지 않도록 제한
