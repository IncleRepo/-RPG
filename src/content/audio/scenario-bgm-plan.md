# 시나리오 기반 BGM 설계 및 제작안

## 문서 목적

이 문서는 `반향해 연대기`의 시나리오 전편을 읽은 뒤,
챕터별 감정선과 장면 전환에 맞춰 어떤 BGM이 필요한지 정리하고
실제 제작된 오리지널 음원 파일의 배치 기준까지 함께 남긴 문서다.

이번 작업 범위는 아래를 포함한다.

- 시나리오 / 설정 문서 전체 검토
- 장면별 필요 음악 카테고리 확정
- 오리지널 루프형 음원 14곡 제작
- 프로젝트 내부 관리용 파일 구조 정리

## 음악 설계 원칙

- 장면의 감정선보다 앞서 나가지 않고, 대사와 컷신의 여백을 남긴다.
- `몽환적 서정성`과 `시간 재난의 불길함`을 메인 톤으로 유지한다.
- 각 챕터는 탐험, 긴장, 전투, 감정 장면이 모두 있으므로 최소 2곡 이상으로 분리한다.
- 공통 모티프는 유지하되, 지역과 세력의 논리가 달라질 때 조성, 리듬, 악기 질감도 함께 바뀌게 한다.
- 특정 작품의 멜로디나 코드 진행을 직접 모사하지 않고, 프로젝트용 패턴으로 독자 제작한다.

## 공통 모티프 방향

- 메인 모티프는 `잃어버린 새벽 서곡`에서 처음 제시한다.
- 이 모티프는 프롤로그 미스터리, 최종 결전, 엔딩 곡에서 리듬과 조성만 바꿔 다시 등장한다.
- 사야 렌과 관련된 장면은 너무 달지 않은 호흡의 리드와 벨 계열 음색으로 정리한다.
- 라오 템 축 장면은 리듬과 저역을 더 세워 현장성과 추진력을 준다.
- 유라 베인과 카델 축 장면은 기계적 반복과 정돈된 박으로 통제감을 만든다.

## 파일 구조

- 실제 음원: `assets/audio/`
- 오디오 매니페스트: `assets/audio/manifest.json`
- 시나리오 기반 기획 문서: `src/content/audio/scenario-bgm-plan.md`
- 코드 참조용 매니페스트: `src/scripts/soundtrack-library.js`
- 재생성 스크립트: `tools/audio/generate_soundtrack.py`

## 제작 및 관리 메모

- 모든 트랙은 프로젝트 전용 루프 BGM으로 직접 생성했다.
- 샘플레이트는 `16000Hz`, 포맷은 `ogg`다.
- 루프 길이는 장면 반복 재생을 고려해 22초에서 58초 사이로 맞췄다.
- 공통 모티프를 공유하지만 트랙별 모드, 템포, 질감을 바꿔 장면 중복감을 줄였다.
- 향후 실제 런타임 연결 시 `assets/audio/manifest.json` 또는 `src/scripts/soundtrack-library.js`를 기준으로 매핑하면 된다.

## 재생성 방법

```bash
python tools/audio/generate_soundtrack.py
```

재생성 시 `assets/audio/*.ogg`와 `assets/audio/manifest.json`이 같은 규격으로 다시 생성된다.
